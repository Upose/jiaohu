Eagles之词频统计

一. 词频统计的优势
       词频--反转文件频率，是一种用于情报检索与文本挖掘的常用加权技术，用以评估一个词对于一个文件或者一个语料库中的一个领域文件集的重要程度。
       简单说来，词频统计也是数据分析的一种形式，通过处理大量数据的词频分析结果能指导用户的行为。
       其实Eagles的很多应用场景中都使用到了词频统计功能，聚合分析模块实际上就是对词频的一种应用。要知道，对于Eagles来说，进行词频统计并不是打肿脸充胖子。就现在而言，Eagles至少有以下优势：
       内置了多种语言的分词器，目前内置英文、中文、日文、俄文、法文、拼音分词。
       不同的分词器有不同的分词算法，用户可以根据自己的需求选择适合的分词器。
       词典支持自定义，以提升分词的准确率。
       当然，以上这几点仅仅是能进行词频操作的敲门砖，如何基于全文搜索快速完成关键词词频统计，才是Eagles的词频功能的关注重点。

二. 词频分析
       结合舆情分析平台来简析Eagles词频统计在实际中的应用。

       a)基于关键词的标签云


标签云实际上是对关键词词频统计的一种应用场景。
具体查询语句的关键词：
aggs：aggregations
terms：词聚合
field：设置为所有字段
include：即包括，正则表达式\u4E00-\u9FA5是汉字,修饰变量2代表只处理2个字以上的词
exclude:即不包括，可对指定冗余无意义的词进行过滤

{
"size": 0,
"aggs":{
    "tags":{
       "terms":{
            "field":"_all",
           "include":"[\u4E00-\u9FA5]{2,}",
"exclude":[ "好的"]
        }
    }
}
}


       在正确查询后，响应结果会给出对应词key，以及相应词的词频doc_count
       在提供的词频数据基础上，可以得到关键词的统计数据，则可实现类似舆情分析云标签。

       b)基于关键词的民情趋势

       民情趋势也是类似的，实质上还是进行词频的统计，只是对分析结果的不同的处理方式而已

       c)基于范围的正负面评价

{
"query" : {
        "terms" : {
          "_all" :[ "交管局", "交警"]
 }
},
    "aggs" : {
        "senti_score" : {
            "range" : {
                "field" : " senti_score ",
                "ranges" : [
                    { "to" : 0},
                    { "from" : 1}
                ]
            }
        }
    }
}

Eagles在词查询的基础上进行范围查询，然后对范围内数据count出结果。
range:范围，其下可对field字段，to/from具体范围进行细化
可分为{ "to" : X1 },
          { "from" :X1, "to" : X2},
          { "from" : X2 }

除此之外，根据实际应用场景的不同还可以对IPV4地址，时间等进行不同的范围聚合。
例如：
对时间范围聚合
          "ranges": { "to": "now-10M/M" },{ "from": "now-10M/M" }]
        第一个范围代表十个月前的月首之前。 第二个范围代表十个月前的月首之后。
对IP地址范围聚合:
           "ranges" : [{ "to" : "10.0.0.5" },{ "from" : "10.0.0.5" } ]
总之，Eagles提供了丰富的范围查询方法。

       d)基于关键词的驾校热力图

        和之前提到的热力图构建方法一样，需要对数据进行提取整合。 例如：想要通过Eagles对各驾校热点进行热力统计。
        那么，首先就会考虑到对数据进行提取，简单来说，热力图需要地理坐标+命中点数，所以实际中就是需要所有驾校地址+所有驾校报名人数。涉及到人数，那就想到aggs的聚合功能。

{
"aggs":{
    "findschool":{
        "terms":{
            "field":"driveschool"
            }

        }
    }
}

       e)基于时间段的热词统计

        数据分析往往具有时效性，对于实时的搜索引擎Eagles来说，对过时数据的把握能力很是重要。其实在实际运用中，Eagles对范围时间内的热词同样能做到精确的查询。
        这里需要运用到时间聚合，对于date字段，可以使用date_range对特定字段进行时间上的筛选。

{
"aggs":{
    "timedata":{
        "date_range":{
            "field":"time",
            "ranges":[
                {
                "from":"now-7d",
                "to":"now"
                }
                      ]
                }
          }
    }
}
