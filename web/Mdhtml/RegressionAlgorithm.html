回归算法

1. 算法介绍

简单来说，回归是一种用来预测的算法模型。一个简单的例子，假设体重（结果）与身高（自变量）间存在一定的关系，这个关系可以用一个函数表示出来：

体重 = a*身高 + b

这就是一个简单的线性回归模型，表示的是身高和体重之间的关系，意思就是知道一个人的身高，就可以算出他的体重：

此处的体重为预测结果，a为系数（权重），b是误差修正；假设a的值为0.8，b的值为5，此时我们可以得到：

  • 当一个人的身高是175cm时，他的体重 = 0.8*170 + 5 = 141；
  • 一个人的体重跟身高是正相关的，也就是说，身高越高，体重越重；身高对于一个人的体重影响最大；

上面是回归算法中，线性回归的一个简单的例子；所谓线性回归，意思就是这个方程的图像是一条直线：

当然，实际中，对于体重，我们还要考虑很多其他因素（自变量）：性别、年龄等，每个自变量都会有相对应的参数（例子中的a），每个参数的权重不同，所以可以得到下面的这个函数：

体重 = a * 身高 + b * 性别 + c * 年龄 + d；

第一个函数称为一元线性回归（一个自变量），第二个函数称为多元线性回归（多个自变量）；当函数的图像画出来不是直线时（通常为曲线或者抛物线），此时称为指数回归或逻辑回归。

图中的黑色点为数据集；红线为线性回归；绿线为非线性（指数或逻辑）回归。
